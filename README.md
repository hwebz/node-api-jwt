# Nodejs Express ResfulAPI with JWT

A ResfulAPI with JWT using Nodejs and Express

# Prerequisites
- Node
- NPM
- Git
- MongoDB
- Express

## Commands
For install dependencie
> npm install

For start dev server on local (Nodejs)
> npm start

## Routes
Register
> /api/user/register

Login
> /api/user/login

Posts
> /api/posts
